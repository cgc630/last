 git 千万别轻易该别人代码，别随便动别人的文件夹
    操作：
	命令行操作：前提有git账号  github  gitlab
	可视化工具
	创建仓库 1、start a project   2、New     public公共 免费 个人选用基本是公共  private私有 收费 美元
        仓库名称  repository name
	last
	cd 项目目录
	git init 初始化 项目和git建立联系
	 针对第一次使用git：git config --global user.name‘用户名’git config --global user.password‘密码’
	git status 查看文件状态
	git add .文件1 文件2 添加到暂存区 单个文件 ：git add . 添加所有
	git commit -m ‘第次提交’ 提交
	git remote add origin master （git仓库地址）链接远程仓库
	git push  -u origin master 第一次提交  推送
	git push（git仓库地址）链接远程仓库    除了第一次提交之外的所有提交都采用    推送
       
	git pull 拉取
	master 主分支
	git branch 显示本地所有分支
	git branch -r 显示远端分支
	git branch -a 显示本地和远端所有分支
	git branch -d 删除本地分支
	git checkout“分支名称” 切换到当前分支
	git fetch -p 更新所有分支
	soucetree 资源树
	git version 查看git版本
    如何创建分枝：
	git branch“分支名称” 创建分支
	git checkout -b “分支名称”   创建并切换到当前分支
    为什么创建分支：
    master 所有内容
